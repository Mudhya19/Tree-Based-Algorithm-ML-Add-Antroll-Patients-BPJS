Bagian B: Integrasi Aplikasi Web (Streamlit) 
Buatlah sebuah aplikasi Streamlit sederhana (app.py) yang memungkinkan pengguna 
berinteraksi dengan model Anda. 
1. Judul Aplikasi: # Tree-Based Algorithm ML Analysis for BPJS Antrol Patients. 
2. Muat Model: Muat file .pkl yang telah Anda simpan dari Bagian A. terdapat dalam folder output format Gradient_Boosting_model.pkl
3. Input Pengguna: 
o Buat form atau widget di sidebar Streamlit untuk menerima input dari 
pengguna. 
o Input ini harus sesuai dengan fitur-fitur yang dibutuhkan oleh model Anda 
o lalu tambahkan sesuai kebutuhan
4. Tombol Prediksi: Buat sebuah tombol (misal: "Prediksi" atau "Analisis Klasifikasi"). 
5. Tampilkan Hasil: 
o Ketika tombol ditekan, ambil semua input dari pengguna, lakukan preprocessing 
yang sesuai, dan gunakan model untuk membuat prediksi. 
o Tampilkan hasil prediksi dengan jelas kepada pengguna.
